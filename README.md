# Emotion Mirror

Real-Time Facial Emotion Recognition Dashboard built using Flask, DeepFace, HTML, CSS, and JavaScript.

## Features

* Real-time webcam-based emotion detection
* Detection of Happy, Sad, Angry, Surprised, and Neutral emotions
* Confidence score visualization
* Dominant mood tracking
* Mood stability analysis
* Detection counter
* Modern dashboard UI

## Technologies Used

* Python
* Flask
* DeepFace
* OpenCV
* HTML
* CSS
* JavaScript

## Project Workflow

1. User grants webcam access.
2. Browser captures live video frames.
3. Frames are sent to Flask backend.
4. DeepFace analyzes facial expressions.
5. Emotion and confidence score are returned.
6. Dashboard updates in real time.

## How to Run

1. Create a virtual environment.
2. Install dependencies.
3. Run:

python app.py

4. Open:

http://127.0.0.1:5000
